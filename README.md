# danielwiedemann.github.io

Personal GitHub Pages user site.

Currently used to host **`.well-known/assetlinks.json`** — the Digital
Asset Links file that links the
[World Cup Schedule](https://github.com/DanielWiedemann/world-cup-2026)
Android TWA to its PWA origin (`https://danielwiedemann.github.io`).
Without this file at the right URL the Android app would show a Chrome
URL bar inside the TWA instead of running fullscreen.

See: <https://developer.android.com/training/app-links/verify-android-applinks>
